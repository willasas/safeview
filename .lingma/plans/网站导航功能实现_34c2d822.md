# 网站导航功能实现

## 概述
在项目中添加网站导航功能,用户可以浏览和访问各类AI工具网站。采用卡片式布局展示,支持分类标签和统计数据。

---

## Task 1: 定义导航数据结构

**文件**: `lib/nav-data.ts`

**任务内容**:
- 创建 `NavItem` 接口定义数据结构
- 包含字段:
  - `name`: 网站名称 (string)
  - `description`: 简短描述 (string)
  - `url`: 网站链接 (string)
  - `icon`: 图标 - 支持 emoji 或 SVG 路径 (string)
  - `tags`: 分类标签数组 (string[])
  - `stats`: 可选统计数据 `{ views?: number; likes?: number; }` (可选)
- 导出导航数据数组 `NAV_DATA`
- 添加 8-10 个示例数据(参考截图中的AI工具)

**示例数据**:
```typescript
export interface NavItem {
  name: string;
  description: string;
  url: string;
  icon: string; // emoji 或 svg 路径
  tags: string[];

    views?: number;
    likes?: number;
  };
}
```

## 技术要点

### 状态管理
- 搜索关键词: `useState<string>`
- 选中标签: `useState<string | null>`
- 显示数量: `useState<number>` (初始8,每次+8)

### 数据过滤逻辑
```typescript
const filteredData = NAV_DATA.filter(item => {
  const matchSearch = !searchTerm ||
    item.name.toLowerCase().includes(searchTerm.toLowerCase()) ||
    item.tags.some(tag => tag.toLowerCase().includes(searchTerm.toLowerCase()));

  const matchTag = !selectedTag || item.tags.includes(selectedTag);

  return matchSearch && matchTag;
});

const displayedData = filteredData.slice(0, displayCount);
```

### 标签颜色方案
根据标签类型自动分配颜色:
- AI对话工具 → blue
- AI写作工具 → green
- AI视频工具 → purple
- AI学习网站 → orange
- 其他 → gray
