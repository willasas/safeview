# UI/UX Design Skill

## 适用场景

当你需要设计用户界面、优化用户体验或创建视觉设计规范时。

## 设计原则

### 1. 用户为中心

- 理解目标用户群体
- 分析用户行为和痛点
- 简化操作流程，减少认知负担
- 提供清晰的反馈和引导

### 2. 一致性

- 统一的色彩系统
- 一致的排版和间距
- 标准化的组件库
- 统一的交互模式

### 3. 可访问性 (Accessibility)

- 足够的颜色对比度（至少 4.5:1）
- 支持键盘导航
- 为所有图片添加 alt 文本
- 使用语义化 HTML

### 4. 响应式设计

- 移动优先策略
- 断点设计：sm(640px), md(768px), lg(1024px), xl(1280px)
- 触摸友好的交互元素（最小 44x44px）
- 自适应布局和字体

## 设计流程

### 阶段 1: 研究与发现

1. 竞品分析
2. 用户调研
3. 定义设计目标
4. 创建设计简报

### 阶段 2: 信息架构

1. 用户流程图
2. 站点地图
3. 内容层级
4. 导航设计

### 阶段 3: 线框图

1. 低保真原型
2. 布局规划
3. 功能分布
4. 用户测试

### 阶段 4: 视觉设计

1. 高保真设计稿
2. 设计系统建立
3. 交互动效
4. 设计评审

### 阶段 5: 交付与迭代

1. 设计规范文档
2. 组件库实现
3. 开发协作
4. 持续优化

## 设计系统要素

### 色彩系统

```css

/* 主色调 */
--primary: #3b82f6;
--primary-hover: #2563eb;

/* 辅助色 */
--success: #10b981;
--warning: #f59e0b;
--error: #ef4444;
--info: #06b6d4;

/* 中性色 */
--background: #ffffff;
--foreground: #111827;
--muted: #6b7280;
--border: #e5e7eb;
```

### 排版系统

- 标题：Inter / SF Pro Display
- 正文：Inter / System UI
- 代码：JetBrains Mono / Fira Code
- 字号比例：1.25 (Major Third)

### 间距系统

基于 8px 网格：
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px

### 阴影层次

- sm: 轻微浮起
- md: 卡片默认
- lg: 模态框
- xl: 下拉菜单

## 工具推荐

- Figma - 界面设计
- Adobe XD - 原型设计
- Coolors - 配色方案
- Unsplash - 图片素材
- Lottie - 动画效果
- Stark - 无障碍检查

## 检查清单

- [ ] 所有交互元素有 hover/focus 状态
- [ ] 表单有清晰的错误提示
- [ ] 加载状态有明确指示
- [ ] 空状态有友好引导
- [ ] 移动端体验流畅
- [ ] 性能指标达标（FCP < 1.8s, LCP < 2.5s）
