## 截图技巧

### 浏览器开发者工具截图（推荐）

1. 打开 Chrome/Edge 浏览器
2. 按 `F12` 打开开发者工具
3. 按 `Ctrl+Shift+P` (Windows) 或 `Cmd+Shift+P` (Mac)
4. 输入 "screenshot"
5. 选择 "Capture full size screenshot"（截取完整尺寸截图）

### 其他方式

- **Windows**: 使用 `Win + Shift + S` 截图工具
- **Mac**: 使用 `Cmd + Shift + 4` 截图
- **在线工具**: 使用浏览器扩展如 "GoFullPage"

---

## 图片优化建议

1. **去除敏感信息**：确保截图中不包含个人隐私信息
2. **统一尺寸**：建议所有截图保持相同的宽度（如 1920px）
3. **适当裁剪**：只保留关键内容，去除不必要的空白
4. **压缩优化**：使用 TinyPNG 等工具压缩图片大小
5. **命名规范**：使用小写字母和连字符，如 `image-detection.png`

---

## 微信公众号适配

如果用于微信公众号文章：

1. **图片宽度**：建议 900px - 1200px
2. **文件大小**：单张图片 < 5MB
3. **图片数量**：3-5 张为宜
4. **首图重要**：第一张图要足够吸引人
5. **添加标注**：可以用箭头、文字标注重点功能
